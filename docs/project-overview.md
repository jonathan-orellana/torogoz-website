# Project Overview

This is a static multi-page website for the Lambda Sigma Upsilon Torogoz
Chapter. It uses plain HTML, CSS, and JavaScript and does not require a build
step.

## Main Areas

- Root HTML files contain the site's individual pages.
- `css/` contains shared and page-specific stylesheets.
- `js/` contains reusable JavaScript.
- `assets/images/` contains photographs and general artwork.
- `assets/icons/` contains standalone SVG icons.
- `assets/logos/` contains chapter and fraternity identity graphics.

`css/home.css` supplies the shared site styles. The remaining stylesheets add
page-specific presentation.
